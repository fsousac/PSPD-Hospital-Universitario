# PSPD-Hospital-Universitario
Aplicação de microsserviços simulando acesso a prontuários eletrônicos (HL7/FHIR) de um Hospital Universitário, com autenticação via Keycloak/OAuth2, controle de acesso por perfil (médico/estagiário/pesquisador) e observabilidade via Prometheus + Grafana em cluster Kubernetes real. Projeto da disciplina PSPD — UnB/FCTE.
